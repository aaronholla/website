# Migrating to Compiled

Make sure to follow [installation](/installation) first.

## Styled Components

eplace the imports with `@compiled/core`.

```diff
-import styled from 'styled-components';
+import { styled } from '@compiled/core';

styled.div`
  font-size: 12px;
  color: ${props => props.color};
`;

styled.div({
  fontSize: 12px,
  color: props => props.color,
});
```

There exists a [codemod](https://github.com/facebook/jscodeshift) to streamline this.

```bash
npx jscodeshift --transform node_modules/@compiled/core/dist/codemods/styled-components-to-compiled <path>
```

Where `<path>` is the path to your source code.
If you're using TypeScript make sure to add `--parser=tsx --extensions=tsx` to your `jscodeshift` call.

## Emotion

Replace the imports with `@compiled/core`,
if you're using `@emotion/styled` the migration is the same from above.

```diff
-/** @jsx jsx */
-import { jsx } from '@emotion/core';
+import '@compiled/core';

(props) => (
  <div
    css={{
      color: 'red',
      backgroundColor: props.background,
    }}
  />
)

(props) => (
  <div
-   css={css`
+   css={`
      color: red;
      background-color: ${props.background}
    `}
  />
)
```

There exists a [codemod](https://github.com/facebook/jscodeshift) to streamline this.

```bash
npx jscodeshift --transform node_modules/@compiled/core/dist/codemods/emotion-to-compiled <path>
```

Where `<path>` is the path to your source code.
If you're using TypeScript make sure to add `--parser=tsx --extensions=tsx` to your `jscodeshift` call.

## Current limitations

Conditionally declared CSS rules are not able to be defined at this time but will be [supported in the future](https://github.com/atlassian-labs/compiled/issues/278).

```jsx
<div
  css={{
    color: 'blue',
    ...(props.isPrimary && { fontSize: 15, fontWeight: bold }),
  }}
/>
```
