# @compiled/babel-plugin

Exports a Babel plugin to enabled Compiled in Babel environments.
See [Installation](/installation) for setup information.
Generally you would not install this directly as it is installed with the core library.

```bash
npm install @compiled/babel-plugin --save-dev
```

## Options

```json
{
  "plugins": [
    [
      "@compiled/babel-plugin",
      {
        "nonce": "__webpack_nonce__"
      }
    ]
  ]
}
```

#### nonce: string

Enables CSP support,
read [Security](/security) for more information.
