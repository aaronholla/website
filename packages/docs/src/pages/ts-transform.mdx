# @compiled/ts-transform-css-in-js

```bash
npm i @compiled/ts-transform-css-in-js
```

## default

Exports a TypeScript transform to enabled Compiled in TypeScript compiler environments.
See [Installation](/installation) for setup information.

### Options

```json
{
  "compilerOptions": {
    "plugins": [
      {
        "transform": "@compiled/ts-transform-css-in-js",
        "options": {
          "nonce": "__webpack_nonce__",
          "sourceMap": true
        }
      }
    ]
  }
}
```

#### Nonce

Enables CSP support,
read [Security](/security) for more information.

#### Source map

Turns on source maps.
Don't use this in production else your bundles will blow out!
