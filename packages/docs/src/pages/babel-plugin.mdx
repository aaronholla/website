# @compiled/babel-plugin-css-in-js

```bash
npm i @compiled/babel-plugin-css-in-js
```

## default

Exports a Babel plugin to enabled Compiled in Babel environments.
See [Installation](/installation) for setup information.

### Options

```json
{
  "plugins": [
    [
      "@compiled/babel-plugin-css-in-js",
      { "nonce": "__webpack_nonce__", "sourceMap": true, "minify": true }
    ]
  ]
}
```

#### nonce: string

Enables CSP support,
read [Security](/security) for more information.

#### sourceMap: boolean

Turns on source maps.
Don't use this in production else your bundles will blow out!

#### minify: boolean

Will minify your CSS - will cause builds to be slower but result in smaller bundles.
