# @compiled/core

The primary entrypoint package for Compiled.

```bash
npm install @compiled/core --save
```

## CSS prop

When the `@compiled/core` module is found CSS prop can be used on React elements.

```jsx
import '@compiled/core';

<div
  css={{
    fontSize: 12,
  }}
/>;
```

## styled

Creates a Styled Component.

```jsx
import { styled } from '@compiled/core';

styled.div`
  font-size: 12px;
`;
```
