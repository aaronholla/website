# @compiled/jest-css-in-js

```bash
npm i @compiled/jest-css-in-js
```

## Jest matcher

Used to enable the compiled CSS matcher in Jest.

```jsx
import { toHaveCompiledCss } from '@compiled/jest-css-in-js';

expect.extend({
  toHaveCompiledCss,
});
```

```tsx
expect(element).toHaveCompiledCss(property-name, value, ?options);
expect(element).toHaveCompiledCss({ [propertyName]: value }, ?options);
```

### Options

```jsx
{
  media: 'screen',
  target: ':hover'
}
```

#### Media

Narrows the assertion down to any valid media query - with `@media` ommitted.

#### Target

Narrows the assertion down to any valid CSS selector.
