# @compiled/core

```bash
npm i @compiled/core
```

## styled

Allows us to create Styled Components.

```jsx
import { styled } from '@compiled/core';

styled.div`
  font-size: 12px;
`;
```

## jsx

Enables the CSS prop on React elements.

```jsx
import '@compiled/core';

<div
  css={{
    fontSize: 12,
  }}
/>;
```

## ClassNames

Allows us to generate a `className` without needing to tie it to a specific React element.

```jsx
import { ClassNames } from '@compiled/core';

<ClassNames>
  {({ css }) => <div className={css({ fontSize: 12 })} />}
</ClassNames>;
```
